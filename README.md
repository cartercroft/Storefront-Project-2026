Storefront 2026 is composed of a .NET Core Web API that handles any backend or authentication requests with a Blazor WASM front end.



Roles:

* User:

&#x09;Users have limited functionality. Users are able to manage their account information, search + add items to their cart, and submit orders.

* Seller:

&#x09;Sellers have extended user functionality. Sellers are able to manage their account information, search and add items to their cart, submit orders, and upload their own items to be sold. Sellers account information should include a company name. Sellers are also able to manage their own items. Sellers do not see their own items in the item search.

* Admin:

&#x09;Admins have the most functionality. Admins can search and manage anybody's items and search and manage users and their permissions.



Future ideas:

&#x09;Give admins the ability to update certain content on the website.

&#x09;Create an company layer so that multiple sellers can manage one company's items.
